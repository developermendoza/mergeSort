Merge Sort

Merge Sort is a divide-and-conquer algorithm that works by dividing the input array into smaller halves, sorting each half, and then merging the sorted halves to produce a fully sorted array. It follows the principle of breaking the problem into smaller sub-problems, solving them, and combining the solutions.

Here's a step-by-step explanation of the Merge Sort algorithm:

  1. Divide: Divide the unsorted array into two halves
  2. Conquer: Recursively sort each half
  3. Combine: Merge the sorted halves to produce a single sorted array.
